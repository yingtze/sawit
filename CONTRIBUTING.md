# Contributing

Terima kasih atas minat Anda untuk berkontribusi pada proyek ini.

Panduan singkat:

- Ikuti gaya penulisan PHP (PSR-12) dan gunakan PHP 8.1+
- Buka issue untuk fitur besar atau bug sebelum mengirim pull request
- Gunakan branch feature/* atau fix/* untuk perubahan
- Sertakan deskripsi singkat dan langkah reproduski pada PR
- Jalankan test: `vendor/bin/phpunit` jika tersedia

Kontak: Tim pengembang internal (Didik Sazali)
